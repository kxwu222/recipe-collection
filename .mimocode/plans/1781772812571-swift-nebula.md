# 食谱小程序 v2.0 版本更新说明

## 版本号
v2.0

## 更新日期
2026-06-18

## 主要功能更新

### 1. AI 智能识别食谱
- 新增"文字识别"功能，用户可粘贴菜谱文字，AI 自动提取食材和步骤
- 支持从小红书、下厨房等平台复制的文字内容
- 使用腾讯云 AI 大模型（qwen3.5-flash）进行智能解析
- 解析失败时提供友好的手动输入引导

### 2. 份量智能缩放
- 查看食谱时可调整"几人份"数量（+/- 按钮）
- 根据人数变化自动按比例调整食材用量
- 支持解析多种用量格式：整数、分数、带单位、"适量"等

### 3. 食谱详情页 UI 重构
- 菜名右侧显示烹饪方法、菜系、烹饪时间标签
- 几人份独立成行，带边框容器样式
- 新增"查看原链接"快捷入口
- 笔记区域可编辑，支持自动高度

### 4. 编辑功能优化
- "描述"更名为"笔记"，更符合用户使用场景
- 编辑模式新增笔记编辑区域
- 图片上传卡片高度优化

### 5. Mock 数据更新
- 替换为 3 个真实食谱：海皇粉丝煲、蚵仔煎、客家酿三宝
- 每个食谱配有封面图片
- 包含完整的食材、步骤、笔记和原链接

## 技术变更

### 新增功能
- `wx.cloud.extend.AI` 直接调用 AI 模型，无需云函数
- Mock 数据版本控制机制（v3），自动刷新本地缓存
- 份量缩放算法：`parseQuantity()`、`changeServings()`

### 移除内容
- 移除所有云函数（parseRecipe、parseRecipe2、parseText、testCloud）
- 移除英文菜名（nameEn）
- 移除旧的 8 个 mock 食谱

### 文件变更
- `miniprogram/pages/recipe/recipe.js` — 新增份量缩放逻辑
- `miniprogram/pages/recipe/recipe.wxml` — UI 重构
- `miniprogram/pages/recipe/recipe.wxss` — 新增样式
- `miniprogram/pages/add-recipe/add-recipe.wxml` — 文字识别 UI
- `miniprogram/pages/add-recipe/add-recipe.js` — AI 识别逻辑
- `miniprogram/utils/api.js` — Mock 数据更新、AI 调用
- `miniprogram/utils/i18n.js` — 新增"煎"烹饪方法、"笔记"翻译
- `miniprogram/assets/images/` — 新增 3 张食谱图片

## 审核备注
- 本版本使用小程序端 `wx.cloud.extend.AI` 接口调用 AI 能力
- AI 功能用于解析用户主动粘贴的菜谱文字，不涉及自动采集用户数据
- 所有食谱内容为本地 mock 数据，不涉及外部 API 调用
